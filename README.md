# page-zh-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对特定网站或域名。仓库旨在提供一个简单、结构化的方式，供用户浏览或引用这些静态页面。

## 项目简介

`page-zh-pages` 是一个静态页面集合仓库。所有页面均为独立的 HTML 文件，可直接在浏览器中打开。本仓库不提供动态服务，也不依赖任何后端或数据库。

## 目录说明

仓库目录结构如下：

```
page-zh-pages/
├── index.html          # 页面索引（可选）
├── pages/              # 页面存放目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── README.md           # 本文件
```

- **`pages/`**：存放所有归档的 HTML 页面。
- **`index.html`**：可选，用于列出所有归档页面的链接，方便浏览。

## 页面归档说明

- 所有页面均为纯前端 HTML，不包含服务器端逻辑。
- 页面之间无依赖关系，可单独使用或分享。
- 归档内容可能不定期更新或清理，恕不另行通知。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/page-zh-pages.git
   ```
2. 直接打开 `pages/` 目录下的任意 HTML 文件，或通过 `index.html` 浏览列表。
